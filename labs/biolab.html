Question 3 (5pts)

You should notice that the second line of the file begins with the characters:

<!--

Strings of characters enclosed in the angle brackets (< and >) are called tags in HTML, and they serve as commands to the browser. This particular tag (starting with <!--) indicates a comment. Comments are used by one programmer (or HTML author) to another and are meant to be ignored by the machine that's operating on the file (in this case, the browser). What does the comment on the second line of the file say?
