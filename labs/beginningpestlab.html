This is the first step in a series of labs that will culminate in the development of your own virtual pet implemented using HTML and Tranquility

Question 1 - The Only Question (25pts)

There are several steps to this phase of the project:

  • Locate or create a series of images that represent your pet in various moods, e.g. hungry, happy, sleepy, angry, etc. If you decide later that you want to use a different set of images, you are free to change them.
  • Using Tranquility, create a web page with the image of your pet in its relaxed state, and a button on the page.
  • Update the Tranquility code for your page so that each time the button is clicked, a different one of the mood pictures is shown. The effect of repeated clicks should be to cycle through all of the pictures you have.

For this question, put the link to your webpage titled pestp1.html
